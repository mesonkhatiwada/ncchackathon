Database Reset Tool

This tool will reset your HamroCare database and recreate all tables with the correct structure.

Warning: This action cannot be undone! All existing data including users, rescue reports, and activities will be permanently deleted.

What will be reset:

  • Users table
  • Admins table (default admin will be recreated)
  • Rescue reports table
  • User activities table
